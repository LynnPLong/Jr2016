Compile & Run Commands:
php hourglass.php arg1 arg2

Parameters:
arg1 = integer greater than 1 representing the height of one bulb of an hourglass
arg2 = integer between 0 and 100 inclusive that represents the percentage of total sand
       in the top bulb of hourglass, user may add '%' symbol at end of parameter if they 
       wish but will not effect the script
